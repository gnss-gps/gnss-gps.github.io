# GNSS GPS Co

This repository hosts the website for [gnss-gps.co](https://gnss-gps.co), which documents how to build a GPS receiver using GNSS-SDR and related tools.

## Website

The website is built with GitHub Pages and is available at https://gnss-gps.co.

## Development

To contribute to the website, clone this repository and make changes to the HTML, CSS, and JavaScript files.

The site is published from the `main` branch.
